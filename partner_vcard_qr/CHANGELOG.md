# Changelog

## 18.0.1.0.0 (2024-03-29)

### Initial Release
- Added QR code generation for partners in vCard format
- Support for complete contact information (name, phone, email, address, position, website)
- Integration into partner form with new "QR vCard" tab
- Automatic QR code regeneration when contact data changes
- vCard 3.0 standard compliance
- Czech localization support
